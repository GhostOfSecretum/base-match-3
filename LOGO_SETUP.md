# Настройка логотипа BASE MATCH-3

## Требования к логотипу

- **Размер:** 1200×630 пикселей (соотношение 1.91:1)
- **Формат:** PNG или JPG
- **Имя файла:** `og-image.png`

## Быстрая установка

### Если у вас уже есть изображение:

1. **Используйте скрипт для изменения размера:**
   ```bash
   ./resize_image.sh путь/к/вашему/изображению.png
   ```

2. **Или вручную через командную строку:**
   ```bash
   sips -z 630 1200 ваше_изображение.png --out og-image.png
   ```

### Создание нового изображения

1. **В графическом редакторе (Canva, Figma, Photoshop):**
   - Создайте новый документ размером 1200×630px
   - Используйте дизайн из описания:
     - Текст "BASE MATCH-3" (BASE - белый, MATCH-3 - золотой)
     - Три круглые иконки сверху
     - Синий градиентный фон с эффектами свечения
   - Экспортируйте как PNG
   - Сохраните как `og-image.png` в корне проекта

2. **Проверка размера:**
   ```bash
   file og-image.png
   # Должно показать: PNG image data, 1200 x 630
   ```

## Где используется логотип

После установки `og-image.png` размером 1200×630, логотип автоматически используется в:

1. ✅ Open Graph превью (социальные сети)
2. ✅ Twitter Card превью
3. ✅ Farcaster miniapp карточка
4. ✅ Превью в Base app

Все ссылки уже настроены в `index.html` на использование `og-image.png`.

## После установки

1. Закоммитьте изменения:
   ```bash
   git add og-image.png
   git commit -m "Update logo to BASE MATCH-3 design (1200x630px)"
   git push
   ```

2. Через 1-2 минуты изображение будет доступно по адресу:
   https://base-match-3.vercel.app/og-image.png
