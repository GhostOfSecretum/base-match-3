# Исправление проблемы с OG Image

## Проблема
Файл `og-image.png` не найден на сервере (404 ошибка), что приводит к:
- ❌ OG Image URL недействителен или недоступен
- ❌ Превью в постах не отображается

## Временное решение

Использован существующий Base логотип как временное решение. Файл `og-image.png` создан из `assets/Logotype/Base_lockup_2color.png`.

**Важно:** Это временное решение. Рекомендуется создать собственное изображение 1200x630px.

## Создание правильного изображения

### Требования
- **Размер:** 1200x630 пикселей (соотношение 1.91:1)
- **Формат:** PNG
- **Содержание:**
  - ✅ Заголовок "BASE MATCH-3" (рекомендуется)
  - ✅ Призыв к действию (например, "Play Now")
  - ✅ Логотип Base
  - ✅ Визуально привлекательный дизайн

### Как создать

#### Вариант 1: Использовать ваш дизайн
Если у вас уже есть изображение с "BASE MATCH-3":
1. Откройте в графическом редакторе
2. Измените размер до 1200x630px
3. Сохраните как `og-image.png`
4. Замените временный файл

#### Вариант 2: Онлайн-редакторы
1. **Canva:**
   - Откройте https://www.canva.com
   - Создайте дизайн 1200x630px
   - Используйте Base логотип из `assets/Logotype/`
   - Добавьте текст "BASE MATCH-3"
   - Используйте синий цвет Base (#0052FF)
   - Экспортируйте как PNG

2. **Figma:**
   - Откройте https://www.figma.com
   - Создайте фрейм 1200x630px
   - Импортируйте Base логотип
   - Добавьте текст и стилизуйте
   - Экспортируйте как PNG

#### Вариант 3: Использовать Base брендинг
1. Откройте `assets/Logotype/Base_lockup_2color.png` в редакторе
2. Создайте новый холст 1200x630px
3. Добавьте логотип Base
4. Добавьте текст "MATCH-3" крупным шрифтом
5. Используйте синий фон Base (#0052FF)
6. Сохраните как `og-image.png`

## Замена временного файла

После создания вашего изображения:

```bash
# Замените временный файл
# (просто сохраните ваше изображение как og-image.png в корне проекта)

# Закоммитьте изменения
git add og-image.png
git commit -m "Replace temporary og-image with custom design"
git push
```

## Проверка

После загрузки:

1. **Проверьте доступность:**
   - Откройте: https://base-match-3.vercel.app/og-image.png
   - Должно отображаться ваше изображение

2. **Проверьте Open Graph:**
   - Используйте [Open Graph Preview](https://www.opengraph.xyz/)
   - Введите: `https://base-match-3.vercel.app`
   - Проверьте, что изображение отображается

3. **Проверьте в Base app:**
   - Создайте новый пост с URL приложения
   - Должно появиться превью с вашим изображением
