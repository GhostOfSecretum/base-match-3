# Создание изображений для Base Match-3

## Необходимые изображения

Вам нужно создать следующие изображения и разместить их в корне проекта (рядом с `index.html`):

### 1. icon.png (512x512px)
- **Размер:** 512x512 пикселей
- **Формат:** PNG
- **Использование:** Иконка приложения в Base app
- **Совет:** Используйте Base logo из `assets/The Square/Base_square_blue.png` и добавьте текст "Match-3"

### 2. splash.png (1200x630px)
- **Размер:** 1200x630 пикселей
- **Формат:** PNG
- **Использование:** Экран загрузки при запуске приложения
- **Совет:** Используйте Base logo + название игры на темном фоне (#000000)

### 3. screenshot1.png, screenshot2.png, screenshot3.png (1200x630px каждый)
- **Размер:** 1200x630 пикселей каждый
- **Формат:** PNG
- **Использование:** Скриншоты игры для публикации
- **Совет:** Сделайте скриншоты игры в действии

### 4. hero.png (1200x630px)
- **Размер:** 1200x630 пикселей
- **Формат:** PNG
- **Использование:** Главное изображение для карточки приложения
- **Совет:** Красивый баннер с игрой и логотипом Base

### 5. og-image.png (1200x630px)
- **Размер:** 1200x630 пикселей
- **Формат:** PNG
- **Использование:** Изображение для социальных сетей (Open Graph)
- **Совет:** Похоже на hero.png, но оптимизировано для соцсетей

### 6. embed-image.png (1200x630px)
- **Размер:** 1200x630 пикселей
- **Формат:** PNG
- **Использование:** Изображение для rich embeds в Base app
- **Совет:** Можно использовать тот же дизайн, что и hero.png

## Быстрый способ создания

### Вариант 1: Онлайн-редакторы
1. Откройте [Canva](https://www.canva.com) или [Figma](https://www.figma.com)
2. Создайте дизайн с нужными размерами
3. Используйте Base logo из папки `assets/`
4. Экспортируйте в PNG

### Вариант 2: Использование Base брендинга
- Логотипы доступны в `assets/Logotype/`
- Квадратные иконки в `assets/The Square/`
- Используйте синий цвет Base: `#0052FF`

### Вариант 3: Скриншоты игры
1. Запустите игру локально
2. Сделайте скриншоты в разных моментах игры
3. Обрежьте до нужных размеров (1200x630px)

## После создания

1. Закоммитьте файлы:
```bash
git add *.png
git commit -m "Add images for Base Mini App"
git push
```

2. Vercel автоматически задеплоит изображения

3. Проверьте доступность:
- https://base-match-3.vercel.app/icon.png
- https://base-match-3.vercel.app/splash.png
- и т.д.

- Или используйте Base logo на однотонном фоне
