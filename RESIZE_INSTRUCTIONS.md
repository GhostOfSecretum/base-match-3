# Инструкция по изменению размера изображения

## Быстрый способ

Если у вас уже есть изображение с "BASE MATCH-3", выполните:

```bash
./resize_image.sh путь/к/вашему/изображению.png
```

Например:
```bash
./resize_image.sh ~/Downloads/my-base-match3-image.png
./resize_image.sh image.jpg
./resize_image.sh assets/my-image.png
```

Скрипт автоматически:
1. Изменит размер до 1200x630px
2. Сохранит как `og-image.png` в корне проекта
3. Сохранит пропорции (может добавить черные полосы, если соотношение сторон отличается)

## Альтернативный способ (вручную)

### Используя sips (macOS):

```bash
sips -z 630 1200 ваше_изображение.png --out og-image.png
```

### Используя онлайн-инструменты:

1. **Canva:**
   - Загрузите ваше изображение
   - Измените размер холста до 1200x630px
   - Сохраните как PNG

2. **Figma:**
   - Импортируйте ваше изображение
   - Создайте фрейм 1200x630px
   - Разместите изображение и обрежьте/масштабируйте
   - Экспортируйте как PNG

3. **ImageMagick** (если установлен):
```bash
convert ваше_изображение.png -resize 1200x630^ -gravity center -extent 1200x630 og-image.png
```

## После изменения размера

1. Проверьте файл:
```bash
file og-image.png
# Должно показать: PNG image data, 1200 x 630
```

2. Закоммитьте изменения:
```bash
git add og-image.png
git commit -m "Update og-image with custom design (1200x630px)"
git push
```

3. Подождите 1-2 минуты, пока Vercel задеплоит

4. Проверьте:
   - https://base-match-3.vercel.app/og-image.png
   - Должно отображаться ваше изображение правильного размера

## Важно

- **Размер:** 1200x630 пикселей (обязательно!)
- **Формат:** PNG (рекомендуется)
- **Соотношение сторон:** 1.91:1 (стандарт для Open Graph)

Если ваше изображение имеет другое соотношение сторон, скрипт добавит черные полосы или обрежет изображение.
